# 🎓 Student Productivity & Performance Dashboard

## 📊 Project Overview
This Power BI dashboard analyzes student productivity and performance using study habits, sleep patterns, mental health, and exam scores.

## 📁 Dataset
The dataset includes:
- Study Hours
- Sleep Hours
- Mental Health Score
- Exam Score
- Gender
- Part-time Job Status
- Productivity Score

Dataset available in the `data` folder.

## 📌 Key KPIs
- Average Study Hours
- Average Sleep Hours
- Average Productivity Score
- Average Mental Health Score

## 📈 Insights
- Productivity peaks around 9 study hours.
- Higher mental health scores correlate with better performance.
- Sleep between 7–9 hours shows optimal productivity.
- Students with balanced study and rest perform better.
